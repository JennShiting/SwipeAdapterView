# SwipeAdapterView

 SwipeAdapterView是一个通过ViewGragHelper实现的能左右滑动的控件，类似tinder APP，左滑表示不喜欢，右滑表示喜欢。

它基于SwipeCardView进行优化改进而实现，通过ViewGragHelper实现左右滑动，解决了SwipeCardView中滑动与点击事件冲突的问题，还增加了子视图复用缓存，提升了控件的性能。
